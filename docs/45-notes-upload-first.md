# 45-笔记生成页网页上传功能文档

## 1. 目标
在笔记生成页增加“直接上传来源文件”功能，并作为首选入口。

## 2. 范围
- 前端：上传区域、文件列表、上传后进入生成链
- 后端：最小上传接口与资产登记

## 3. 功能要求
- 支持上传：音频 / 视频 / 图片 / 文本 / Markdown / PDF
- 上传成功后自动进入 SourceAsset 登记
- 在笔记生成页的视觉优先级高于“扫描目录”
- 管理员可上传，普通用户禁用

## 4. 接口要求
- 新增最小可用上传接口（例如 `POST /api/v1/sources/upload`）
- 接口返回已登记的 SourceAsset 记录
- 不替换原有扫描目录功能，而是作为首选路径并存

## 5. 验收标准
- 可直接通过网页上传文件
- 上传后能在笔记生成页看到并继续生成
