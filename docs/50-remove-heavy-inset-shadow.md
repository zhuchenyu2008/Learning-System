# 50-织物风格内阴影收敛修整文档

## 1. 用户反馈
用户明确反馈：**不要轻微内阴影，当前很多区块/控件都带内阴影，视觉上很难受。**

## 2. 修整目标
- 收掉大面积、普遍性的内阴影用法
- 保留织物质感，但改用更柔和的：
  - 纹理
  - 缝线
  - 外阴影
  - 柔光
  - 边框高光
- 避免所有控件都出现同类 inset 视觉语言

## 3. 范围
- `styles.css` 中 fabric 系列样式
- 受影响明显的卡片/按钮/输入框/上传区块/设置控件
- 不改业务逻辑

## 4. 验收标准
- 内阴影被明显收敛或移除
- 织物质感仍保留
- build 通过
