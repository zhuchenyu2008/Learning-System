# 97-新一轮后测问题主文档

## 1. 用户新反馈
在当前最新环境人工测试后，用户新增反馈：
1. 复习卡片虽然有独立任务，但生成/完成时间几乎只差几秒，怀疑根本没走 AI
2. 知识点总结没有预览
3. 生成的思维导图非常混乱

## 2. 本轮目标
围绕上述 3 个问题，先形成最小文档与任务拆分，再进入实现/验证。

## 3. 范围
- 复习卡片生成链是否真正走 LLM / 是否只是规则切片
- summary 产物预览链为何缺失
- mindmap 产物质量与 prompt / sanitize / 渲染链问题

## 4. 非目标

- 不扩张到完整新产品设计
