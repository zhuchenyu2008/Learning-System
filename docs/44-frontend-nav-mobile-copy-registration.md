# 44-前端导航与移动端修整文档

## 1. 范围
- 侧边栏结构重做
- 移动端适配
- 文案收敛
- 注册入口补齐

## 2. 导航结构要求
当前不要两层侧边栏。应改为：
- 一级分组：笔记 / 复习 / 设置
- 每个一级分组下直接展开/收起子项：
  - 笔记：总览 / 笔记生成 / 笔记库
  - 复习：总览 / 复习 / 知识点总结 / 思维导图生成
  - 设置：AI 配置 / 工作区与 Obsidian / 用户与登录情况 / 导入导出 / 任务与调度

## 3. 移动端要求
- 支持抽屉式侧边栏或顶部菜单切换
- 主内容区在窄屏下单列显示
- 表单/卡片/按钮适应窄屏宽度
- 三栏页面在移动端需退化为单列/分段式布局

## 4. 文案要求
- 去掉多余解释性说明
- 默认结论优先，说明从简
- 保留必要的字段说明与错误提示

## 5. 注册入口要求
- 当系统允许注册时，登录页必须出现注册入口
- 若系统禁止注册，则入口隐藏或置灰说明
