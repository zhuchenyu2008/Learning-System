# 86-后测问题拆解文档

## 1. 笔记内容污染
### 现象
最终笔记中混入不应暴露给用户的过程性/检索性/提示性内容。
### 重点排查
- 结构化生成输出如何拼进 Markdown
- retrieval 摘要是否错误地写进正文主体
- warnings / metadata / context 是否误入最终笔记

## 2. 笔记生成页提示文案错误
### 现象
创建任务后显示“已创建任务，生成0篇笔记”而不是“正在生成”。
### 重点排查
- 前端轮询状态映射
- queued/running/completed 三阶段文案

## 3. docx 失败
### 现象
docx 无法生成。
### 重点排查
- docx 解析
- 新结构化输出链与 docx 路径兼容
- worker/job 失败日志

## 4. 思维导图预览失败
### 现象
思维导图页面渲染失败，同时影响笔记生成页。
### 重点排查
- Mermaid/图数据格式
- 前端渲染组件与接口返回契约
- 生成内容是否包含非法结构

## 5. 思维导图/知识点总结目录不当
### 现象
思维导图和知识点总结与笔记混放。
### 重点排查
- review artifacts 路径策略
- 前后端显示分组

## 6. 复习卡片生成方式不对
### 现象
应在笔记生成后另起任务生成卡片，参考两个 skill。
### 重点排查
- 是否应引入异步 job
- 复习卡片与主笔记生成解耦

## 7. 复习时长统计为 0
### 现象
复习页面统计异常。
### 重点排查
- watch/report 接口
- 前端上报节奏
- 聚合统计逻辑

## 8. 复习时 AI 参与缺失
### 现象
复习时缺 AI 打分与讲解。
### 重点排查
- review session 流程
- 与参考 skill 的判分/讲解逻辑差距
