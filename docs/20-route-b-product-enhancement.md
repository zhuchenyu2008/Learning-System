# 20-路线B：体验优化 / 真功能增强文档

## 1. 前置条件
只有当路线 A 完成并验收后，才进入路线 B。

## 2. 目标
在工程化基础稳住后，进一步提升产品体验、内容质量与真实任务能力。

## 3. 子目标拆解
### B1. 异步任务增强
- 将 notes/review/summaries/mindmaps 的主链逐步切到真实 worker 执行
- 增强任务状态、失败重试、日志可见性

### B2. PDF / 多模态处理增强
- 优化 PDF 页级拆分策略
- 提升多模态模型输入组织质量
- 改善 OCR / 视觉理解结果的稳定性

### B3. 知识点抽取增强
- 更细粒度切片
- 更强的标签、关联、去重
- 更适合后续复习与总结

### B4. 行为埋点增强
- 观看时长
- 页面访问
- 用户活动快照
- 管理员分析视图增强

### B5. UI/交互优化
- 织物质感主题细节强化
- 表单/列表/反馈体验优化
- 错误提示与空态体验优化

### B6. 联调试用与修补
- 基于真实试用反馈修补主链问题
- 做下一轮产品级验收

## 4. 验收标准
- 异步任务体验更真实
- 多模态输入质量更高
- 复习与知识处理结果更稳定
- 用户体验更完整
