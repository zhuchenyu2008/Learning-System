## 3. 建议 subagent 拆分

- 下载两个参考仓库到临时目录

- 复现并定位用户列出的 8 个问题中的关键日志/关键文件

- 基于 SA-42/43 回传，形成最终修复路线建议（此步先由主 agent 汇总，不先派实现）

## 4. 串并行关系

- 主 agent 汇总后再决定是否进入实现型 subagent
