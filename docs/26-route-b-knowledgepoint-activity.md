# 26-路线B知识点抽取与埋点增强文档

## 1. 目标
在现有 notes/review/settings 主链基础上，增强：
- 知识点抽取质量
- 关联/去重能力
- 用户活动与观看时长埋点
- 管理员分析视图的数据基础

## 2. 子目标
### B3.1 知识点抽取增强
- 更细粒度切片
- 更稳定的标题/段落/列表提取
- 同 note 内部去重
- 更适合复习卡生成

### B3.2 关联与元数据增强
- 补知识点标签/来源锚点/摘要字段
- 为后续关联与去重预留更清晰结构

### B3.3 用户活动埋点增强
- 页面访问事件
- 观看时长/停留时长
- review 行为之外的浏览行为记录
- 形成管理员可查看的数据基础

## 3. 验收标准
- KnowledgePoint 主链有质量增强证据
- 至少有一条用户活动埋点链真实落库/可查询
- 不破坏现有 review/settings API 契约
