# SA-22A 第三轮前端体验修整任务单

## 目标
修顶栏标题位置/样式、上传区块和各种控件风格、继续收敛文案。

## 必读文档
- docs/44-frontend-nav-mobile-copy-registration.md
- docs/48-first-feedback-round2-remediation.md

## 用户新增要求（必须对齐）
- 顶栏标题不要放成现在顶部那种独立标题，而要回到“笔记式导航”位置与样式表达
- 新加区块与控件（上传按钮、勾选、允许注册开关、文件选择按钮等）要更贴近织物质感要求
- 继续减少解释性/过时文案

## 范围
- 顶栏标题/导航表现
- 上传区块视觉
- checkbox / file input / toggle 等控件样式
- 文案继续收敛

## 不要做
- 不扩展新业务功能
- 不改后端逻辑

## 验收标准
- 视觉与文案问题有明确收敛证据
