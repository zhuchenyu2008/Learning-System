# SA-20A 前端首轮体验修整任务单

## 目标
修整前端导航结构、移动端体验、页面文案与注册入口。

## 必读文档
- docs/43-first-ux-issues-remediation.md
- docs/44-frontend-nav-mobile-copy-registration.md

## 范围
- 侧边栏改为一级分组可展开/收起
- 移动端适配
- 解释性文字收敛
- 注册入口补齐

## 不要做
- 不扩展新业务功能
- 不改后端接口定义（除注册入口显示所需读取现有配置）

## 验收标准
- 导航结构符合要求
- 移动端基本可用
- 登录页注册入口符合 allow_registration 状态
