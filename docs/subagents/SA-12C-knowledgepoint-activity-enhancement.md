# SA-12C 知识点抽取与活动埋点增强任务单

## 目标
增强知识点抽取质量，并补用户活动/观看时长相关埋点。

## 必读文档
- docs/20-route-b-product-enhancement.md
- docs/25-route-b-orchestration.md
- docs/26-route-b-knowledgepoint-activity.md

## 范围
- 增强 KnowledgePoint 抽取逻辑与元数据
- 补最小用户活动埋点链
- 让 admin 侧有更真实的数据基础

## 不要做
- 不破坏现有 review/settings 契约
- 不扩展到无关业务

## 验收标准
- 有抽取质量增强证据
- 有活动埋点可见证据
