# 01-需求拆解文档

## 1. 用户角色
### 1.1 管理员
- 初始化系统
- 配置 AI / Embedding / STT / OCR / Obsidian / 注册策略
- 管理本地工作目录与导入任务
- 生成/重生成笔记、复习计划、总结、思维导图
- 查看用户登录情况、观看时长、复习日志
- 执行数据库导入/导出

### 1.2 普通用户
- 登录系统
- 查看笔记、复习内容、知识点总结、思维导图
- 进行复习并提交复习日志
- 不可上传/删除/修改笔记及后台配置

## 2. 输入源需求
- 音频：录音文件，需转写。
- 视频：提取音轨后转写，可结合关键帧 OCR / 多模态理解。
- 图片：OCR 或多模态识别。
- 文本：直接解析与清洗。
- 现有 Markdown：扫描入库、切片、索引、关联。

## 3. 输出物需求
- Markdown 笔记
- 知识点条目
- FSRS 复习卡/复习任务
- 知识点总结
- Mermaid 思维导图 Markdown / SVG 预览
- 任务日志与导入/导出包

## 4. 功能需求拆解
### 4.1 笔记模块
1. 总览
   - 文件夹状态
   - 最近导入
   - 最近生成笔记
   - 待处理任务
2. 笔记生成
   - 选择输入源
   - 配置笔记生成策略
   - 配置额外提示词（全局来自设置）
   - 手动触发生成
3. 笔记库
   - 树状目录
   - 标签/来源/类型筛选
   - 笔记详情渲染（HTML + Mermaid）

### 4.2 复习模块
1. 总览
   - 今日待复习量
   - FSRS 统计
   - 最近复习情况
2. 复习
   - 逐条复习
   - 打分（Again/Hard/Good/Easy）
   - 写入复习日志
3. 知识点总结
   - 手动生成
   - 定时生成
   - 可选笔记范围
   - 可选额外提示词
4. 思维导图生成
   - 手动生成
   - 定时生成
   - 可选笔记范围
   - 可选额外提示词
   - Mermaid 输出

### 4.3 设置模块
- AI 配置
- Embedding 配置
- STT 配置
- OCR / 多模态配置
- Obsidian 配置
- 文件夹配置
- 注册开关
- 用户列表/登录情况/观看时长
- 导入导出数据库
- 定时任务策略

## 5. 非功能需求
- 可部署：Docker Compose
- 可扩展：标准 REST API
- 可维护：模块化代码结构
- 安全：RBAC + 输入校验 + 路径白名单
- 可观测：任务状态、错误日志、用户行为日志

## 6. 关键约束
- AI 配置必须填写后才能使用相关功能。
- Embedding / STT / OCR 配置必须独立可用。
- 普通用户界面保持一致，只显示灰色禁用。
- 数据与笔记文件必须尽量可导出、可迁移。
