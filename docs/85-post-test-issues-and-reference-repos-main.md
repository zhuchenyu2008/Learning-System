# 85-后测问题与参考仓库对标调查主文档

## 1. 用户新反馈
用户在最新 Docker 环境人工测试后，反馈以下问题：
1. 笔记内容不纯粹，夹杂一大段不应进入最终笔记的内容
2. 笔记生成页提示文案不对：当前显示“已创建任务，生成0篇笔记”，应改为“已创建任务，正在生成笔记”，完成后提示“笔记已生成”
3. docx 无法生成笔记，需要看失败日志
4. 思维导图生成后未正常预览，页面底部提示渲染失败，并在笔记生成页面也导致报错
5. 思维导图和知识点总结不应与笔记混在一起，应单独列文件夹
6. 复习卡片应在笔记生成后另起任务，根据笔记内容生成卡片，参考两个 skill
7. 复习页面时长统计一直为 0
8. 复习时应有 AI 参与：AI 打分并讲解

## 2. 用户追加要求
- 将参考的两个仓库下载到临时目录
- 仔细研究、对比当前项目与参考实现的差异
- 基于真实问题重新形成问题清单、差距清单与解决方案

先做“参考仓库对标 + 当前项目问题复盘 + 方案文档化”，再进入实现。
