# 09-前端预览启动任务文档

## 1. 目标
按用户要求将前端在本机启动到 `18794` 端口，供用户预览当前界面。

## 2. 前置条件
- 前端依赖已安装
- 前端 `npm run build` 已通过
- 当前只做预览启动，不扩展业务功能

## 3. 范围
- 在 `frontend` 目录启动 Vite 开发服务器
- 绑定 `0.0.0.0:18794`
- 验证端口监听与本地 HTTP 返回

## 4. 不做的事
- 不改业务代码
- 不改后端
- 不进入 SA-02/SA-03

## 5. 验收标准
- 18794 端口启动成功
- 本机访问返回 HTML
- 回传启动命令与验证结果
